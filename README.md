# RecyclerView Project

پروژه ساده درس موبایل 2 برای نمایش لیست دانشجوها با RecyclerView.

## بخش‌های اصلی

- `MainActivity.java`: ساخت داده نمونه، تنظیم `LinearLayoutManager` و اتصال Adapter
- `Student.java`: مدل داده هر دانشجو
- `StudentAdapter.java`: کلاس Adapter و ViewHolder برای RecyclerView
- `activity_main.xml`: صفحه اصلی شامل عنوان و RecyclerView
- `item_student.xml`: ظاهر هر آیتم لیست

## اجرا

پوشه پروژه را در Android Studio باز کنید و بعد از Sync، برنامه را روی Emulator یا گوشی اجرا کنید.
